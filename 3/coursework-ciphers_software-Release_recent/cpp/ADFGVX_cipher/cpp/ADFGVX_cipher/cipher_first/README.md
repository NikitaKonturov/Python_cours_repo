## Описание проекта ADFGVX_cipher

### Условия работы функций
  
- **`encript()`**:
  - В ключевом множестве ключи между собой должны быть разделены знаком '|'. Его можно изменить
  
- **`columnar_transposition_encrypt()`**:
  - Если прямоугольник неполный, тогда дополняется символом 'X', который можно будет изменить.

- **`decript()`**:
  - Для Извлечения из ключевой строки транспозиционного ключа. Ожидаемый формат: <отформатированная таблица>\nТранспозиционный  ключ:ключ>
